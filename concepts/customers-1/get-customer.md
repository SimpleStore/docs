# Get Customer

Retrieves a Customer by the Unique Customer Identifer. 

### **GET**

```text
/v1/customers/{CustomerId}
```

#### Request: Route

| Element | Type | Description |
| :--- | :--- | :--- |
| CustomerId | guid/uuid | A unique identifier for the customer |

#### Request: Query

No Query

#### Request: Body

No request body

#### Response: 200 Success

See [Customer](./)
