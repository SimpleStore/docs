# Get Cart By Cart Id

Retrieves the Cart by the Unique Cart Identifer. 

### **GET**

```text
/v1/cart/{CartId}
```

#### Request: Route

| Element | Type | Description |
| :--- | :--- | :--- |
| cartId | guid/uuid | A unique identifier for the cart |

#### Request: Body

No request body

#### Response: 200 Success

Cart Object \(See [Cart](./)\)
